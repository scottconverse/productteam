2. What ProductTeam Is

ProductTeam is a set of AI agent skills for product development. Each skill is a SKILL.md file that defines an agent's role, process, and rules. The skills are designed for use with Claude Code in the Claude desktop app.

  • PRD Writer — Takes a product concept, asks clarifying questions, researches competitors, and produces a structured PRD.
  • Planner — Reads a PRD and decomposes it into sprint contracts with testable acceptance criteria.
  • Builder — Implements sprint contracts with production-quality code and tests.
  • UI Builder — Specialized builder for visual work: landing pages, dashboards, web UIs.
  • Evaluator — Verifies code against sprint contracts, runs tests, tries to break things, and grades PASS/NEEDS_WORK/FAIL.
  • Design Evaluator — Grades visual artifacts on Coherence, Originality, Craft, and Functionality.
  • Doc Writer — Reads code and produces README, landing page, PDF, changelog, and terms of service.
  • Orchestrator — Routes work between agents, manages build-evaluate loops, and enforces approval gates.

ProductTeam is not a standalone tool. It requires Claude Code as the runtime environment. The skills produce suggestions, code, and documentation that require human review before use.
